# 홍채 색 인식 프로젝트

딥러닝과 이미지 처리 기법을 활용하여  
홍채 영역을 인식하고, 홍채 색을 분석하는 프로젝트입니다.

---

## 📌 프로젝트 개요
- 홍채 이미지를 기반으로 홍채 영역을 탐지
- 추출된 영역의 색상을 분석하여 홍채 색 분류
- CNN 모델을 활용한 이미지 분류 수행

---

## 🛠 사용 기술
- Python
- TensorFlow / Keras
- OpenCV (cv2)
- NumPy
- Matplotlib

---

## 📊 주요 기능
- 이미지 전처리 (그레이스케일 변환, 노이즈 제거)
- HoughCircles 기반 홍채 영역 탐지
- CNN 기반 홍채 색 분류
- 학습/검증 데이터 기반 모델 성능 평가

---

## 📁 파일 구성
- iris_detection.ipynb
- iris_color_classification.ipynb
- eyes.zip
- etc
---

## 💡 프로젝트 특징
딥러닝 모델(CNN)과 이미지 처리(OpenCV)를 함께 활용하여  
홍채 영역 인식과 색상 분석을 동시에 수행한 프로젝트입니다.

---

## 📊 결과
- 홍채 영역을 원 형태로 탐지
- 색상 정보를 추출하여 홍채 색 구분 가능
- 학습 과정에서 손실 및 정확도 변화 확인
